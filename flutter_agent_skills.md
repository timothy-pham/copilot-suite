# 📦 Flutter Agent Skills

Tổng hợp các **Agent Skills cho Flutter Development**, bao gồm bộ chính thức (Official) và bộ tùy chỉnh (Custom) giúp tăng tốc phát triển, chuẩn hóa kiến trúc và tối ưu hiệu năng.

---

# 🚀 Official Skills

> Những bộ kỹ năng tiêu chuẩn, đảm bảo tính ổn định và quy trình phát hành chuyên nghiệp.

| Skill                                | Mô tả                                                                                     | Link                                                                                        |
| ------------------------------------ | ----------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------- |
| **Flutter Official Skills**          | Bộ kỹ năng cốt lõi giúp tối ưu hóa SDK, quản lý đa ngôn ngữ (i18n) và hiệu năng runtime.  | https://github.com/flutter/skills                                                           |
| **Frontend Design**                  | Nâng cấp thẩm mỹ UI/UX cao cấp, loại bỏ cảm giác “code AI” thô cứng.                      | https://github.com/anthropics/skills/tree/main/skills/frontend-design                       |
| **Architecture Blueprint Generator** | Tự động phân tích và ép buộc code tuân thủ kiến trúc chuẩn (Clean Architecture, Modular). | https://github.com/github/awesome-copilot/tree/main/skills/architecture-blueprint-generator |
| **Apple App Store Reviewer**         | Kiểm tra code theo guideline Apple để tránh bị reject khi publish.                        | https://github.com/github/awesome-copilot/tree/main/skills/apple-appstore-reviewer          |
| **Agentic Eval**                     | Tự động đánh giá logic, kiểm tra SOLID trước khi trả output cuối.                         | https://github.com/github/awesome-copilot/tree/main/skills/agentic-eval                     |
| **Add Educational Comments**         | Tự động viết comment giải thích logic ("why") giúp maintain lâu dài.                      | https://github.com/github/awesome-copilot/tree/main/skills/add-educational-comments         |
